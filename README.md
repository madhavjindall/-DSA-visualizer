# DSA Visualizer (C++)

A console-based application to visualize how common data structures and algorithms work step by step.  
The goal of this project is to make it easier to understand the internal working of algorithms by showing each operation as it happens.

---

## Features

- Visualization of Bubble Sort  
- Visualization of Selection Sort  
- Step-by-step tracing of Binary Search  
- Clear console output for each operation  
- Delays added to simulate execution flow  
- Highlighting of elements during comparisons  

---

## Tech Stack

- C++
- Standard Template Library (STL)
- Console-based interface
